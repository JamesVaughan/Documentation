# **Export Network Tables**
This tool allows the exporting of up to five CSV tables from the selected scenario. Table columns are limited to "standard", "extra" and "result" categories as defined in the Network API (i.e., link mode codes are omitted).


## **Using the Tool with Modeller**
The tool can be found in "TMG Toolbox" -> "Analysis" -> "Export Network Tables"

### Scenario
Select the scenario to execute against. Only one scenario at a time can be selected.

### Folder
Select "Browse..." to navigate to the target folder to store the output CSVs.

### File Prefix
(Optional) Enter the text to prepend to export files. Default file names are "links.csv", "nodes.csv", etc.

### Flags for Network Elements
Choose the type of the network elements to be exported (select all that apply).


## **Using the Tool with XTMF**
The tool is not callable from XTMF. Please use Emme Modeller.
